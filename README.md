## Building
0. Install [Hugo]()
1. Clone this repository (`git clone https://github.com/doamatto/srhs-astroclub.git`)
2. Build the site (`hugo`), or serve it locally (`hugo server -wv`) and open it locally ([http://localhost:1313](http://localhost:1313))
